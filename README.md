# RCSceneGameRoom

RCSceneGameRoom 是融云（RongCloud）场景化 SDK 体系中的游戏语音房模块，基于融云 RCVoiceRoomLib、RCTinyGameLib 及 RCSceneRoom 基础层构建，为 iOS 应用提供"语音连麦 + 实时小游戏"一体化的娱乐房间解决方案。
